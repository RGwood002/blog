## Project overview

Hardcover has offered an api that allows for users to control their data outside of their site. It has spawned a wonderful amount of projects from the community.

They also have a mobile app that allows for you to scan a book into your account via barcode! I do enjoy this feature very much. So much so that I find myself just
navigating the app through this one feature. I enjoy simply scanning in a book, creating an entry that I am either currently reading this book or I want to read this
book. The flow is simple enough that I find it worth the effort to create a system that streamlines this behavior for myself. 

It is also a great excuse to practice frontend development.

## Created with
The project contains the following:
- Jetson nano W/usb camera
- Docker
- Django
- Next.js
